## nginx使用

- mac
    - 启动 ``` nginx ```
    - 退出 ``` nginx -s quit ```
    - 重启 ``` nginx -s reload ```
    - 停止 ``` nginx -s stop ```

- windows
    - 下载，解压 ，进入相应目录
    - 启动 ```start nginx ```
    - 退出 ``` nginx -s quit ```
    - 重启 ``` nginx -s reload ```
    - 停止 ``` nginx -s stop ```
